# Titanic Dataset Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify key factors affecting passenger survival.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Key Steps
- Data cleaning and handling missing values
- Removal of duplicate records
- Data visualization to analyze survival patterns
- Insight generation using statistical analysis

## Key Insights
- Female passengers had a higher survival rate.
- First-class passengers were more likely to survive.
- Higher ticket fares increased survival probability.
